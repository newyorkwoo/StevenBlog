## 🔍 如果無法訪問

### 1. 檢查防火牆設定

**macOS 防火牆：**

```bash
# 查看防火牆狀態
sudo /usr/libexec/ApplicationFirewall/socketfilterfw --getglobalstate

# 如果防火牆開啟，需要允許 Node 連接
# 系統偏好設定 > 安全性與隱私 > 防火牆 > 防火牆選項
# 允許 "Node" 接收傳入連線
```

### 2. 確認 IP 地址

如果 IP 地址改變了，重新獲取：

```bash
# 在終端機執行
ifconfig | grep "inet " | grep -v 127.0.0.1
```

常見的 IP 格式：

- `192.168.x.x` - 家用路由器
- `172.x.x.x` - 企業網絡或熱點
- `10.x.x.x` - 大型網絡

### 3. 重新啟動服務器

如果服務器沒有顯示 Network 地址：

**前台：**

```bash
cd /Users/steven/Documents/myproject/StevenBlog/packages/blog-frontend
npm run dev
```

**後台：**

```bash
cd /Users/steven/Documents/myproject/StevenBlog/packages/blog-admin
npm run dev
```

### 4. 檢查是否在同一網絡

**電腦端查看 WiFi：**

- 點擊菜單欄的 WiFi 圖標
- 查看當前連接的網絡名稱

**手機端查看 WiFi：**

- 設定 > WiFi
- 確認連接的網絡名稱與電腦相同

## 💡 進階技巧

### 使用 QR Code 快速訪問

1. 安裝 qrcode-terminal：

```bash
npm install -g qrcode-terminal
```

2. 生成 QR Code：

```bash
# 前台
qrcode-terminal "http://172.20.10.2:3000/"

# 後台
qrcode-terminal "http://172.20.10.2:5200/"
```

3. 用手機掃描 QR Code 即可訪問

### 固定 IP 地址（可選）

為了避免 IP 變動，可以在路由器設定中為您的 Mac 分配固定 IP：

1. 登入路由器管理介面
2. 找到 DHCP 設定或 IP 綁定
3. 將 Mac 的 MAC 地址綁定到固定 IP
